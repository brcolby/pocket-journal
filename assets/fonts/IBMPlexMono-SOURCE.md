# IBM Plex Mono Bold

`IBMPlexMono-Bold.ttf` is IBM Plex Mono Bold version 2.005. The checked-in
binary has SHA-256
`74e5eedcfa4596497d34e19023cabdabd3a8c852b903007a5654a59591a72ffb`.

It is used only for spaces, ASCII punctuation, and the unsupported-codepoint
fallback in the firmware UI. Letters and numbers come from the checked-in
Carbon sources. Generated files use the IBM Plex name directly; legacy
`Space Mono` aliases are not part of this asset contract.

`IBMPlexMono-LICENSE.txt`.
